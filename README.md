# Task 1 – Data Cleaning and Preprocessing

## Objective
Clean and preprocess a raw dataset to make it structured and analysis-ready using Python and Pandas.

## Dataset
Netflix Movies and TV Shows Dataset from Kaggle.

## Tools Used
- Python
- Pandas
- Google Colab

## Data Cleaning Steps
1. Checked for missing values using `isnull()`.
2. Handled null values using `fillna()`.
3. Removed duplicate rows.
4. Standardized column names for consistency.
5. Converted date columns into datetime format.
6. Fixed inconsistent text formatting (e.g., capitalization, spacing).
7. Verified and corrected data types where necessary.

## Outcome
A cleaned and structured dataset ready for analysis and visualization.

- Raw dataset (`netflix_titles.csv`)
